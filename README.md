# AngryFlappy Turtle Game

AngryFlappy Turtle is a Java and JavaFX game where you guide a turtle through underwater obstacles, avoiding danger and collecting sea shells for points.

## Features

- Flappy style mouse click movement
- Three difficulty levels: easy, medium, hard
- Obstacles, sharks, and ocean floor collisions
- Sea shells for bonus points and a pearl that triggers autopilot

## Requirements

- Java Development Kit (JDK)
- JavaFX SDK

## How to Run

Compile and run the main file from the terminal:

javac AngryFlappyBird.java
