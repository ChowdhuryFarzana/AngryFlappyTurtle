A Java/JavaFX arcade game where you guide a turtle through underwater obstacles, collect sea shells, and survive as long as possible.

**Contributors:** Anh Nguyen, Eris Nguyen, Farzana Chowdhury

## Features

Mouse-click controls, multiple difficulty levels, collision detection with pipes and sharks, collectible sea shells and pearls for autopilot mode, smooth animations and sound effects.

## Getting Started

**Requirements:** Java JDK 11+, JavaFX SDK

**Run:**
```bash
javac AngryFlappyBird.java
java AngryFlappyBird
```

Or use IntelliJ, Eclipse, or VS Code with JavaFX support.

## How to Play

Select difficulty, click "Start game!", click mouse to make turtle rise, avoid obstacles, collect sea shells for points, grab pearls for autopilot. Survive and beat your high score.

## Structure

**AngryFlappyBird.java:** Core game logic, UI, animation, collision detection, scoring  
**Defines.java:** Constants, asset paths, configuration

## Technologies

Java, JavaFX for graphics and animation
